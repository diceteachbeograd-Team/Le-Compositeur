# Release Staging 1.20260306.1

Versioning scheme:
- `1.YYYYMMDD.N`
- `N` starts at `1` each day and increments for additional builds on the same day.

Examples:
- `1.20260306.1`
- `1.20260306.2`
- `1.20260307.1`

Place artifacts in:
- `linux-rpm/`
- `linux-deb/`
- `windows-x86_64/`
- `macos-aarch64/`
- `macos-x86_64/`
